## Installation

1. Clone the repository

```
git clone https://github.com/rvitality/ec-pc.git .
```

2. Install the dependencies

```
cd .\frontend\
npm install
```

3. Run the server. Do note that the server runs on port 3000.

```
npm start
```

## Run backend server

1. Open new terminal
2. Change directory

```
cd .\backend\
```

3. Run flask server

```
flask run
```
